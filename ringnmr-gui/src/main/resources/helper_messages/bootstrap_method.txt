Specifies the means by which datasets are sampled for bootstrapping, in order to derive parameter errors, as well as values if bootstrap aggregation or regularization are used.

Parametric
Each bootstrap dataset comprises datapoints which have been derived from the original data by perturbing their values with noise. Each datapoint's noise value is sampled using a normal distribution whose standard deviation is given by the datapoint's error bar.

Nonparametric
Bootstrap datasets are genreated by sampling from the original dataset with replacement. In effect, nonparametric bootstrapping involves weighting datapoints with integer weights (the sum of all weights will always equal the number of datapoints). There are a finite number of samples that can be generated using this method, depending on the moiety type and the fields that data were aquired at:

The number of bootstrap samples is related to the number of unique ω-values, which is based on the fields used. For a given field, spectral densities at 0, ωD and 2ωD are available. However, during the spectral density mapping, only a single point is generated for each UNIQUE ω value. The most unique ω-values possible for N fields is 2N + 1. Bayesian
Bootstrap datasets are sampled by weighting datapoints using samples from a symmetric Dirichlet distribution.
