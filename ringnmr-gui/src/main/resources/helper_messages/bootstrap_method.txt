Specifies the means by which datasets are sampled for bootstrapping, in order to derive parameter errors, as well as values if bootstrap aggregation or regularization are used.

Parametric
Each bootstrap datapoint is derived from the original by perturbing its value with normally distributed noise, using the datapoint's error bar as the standard deviation.

Nonparametric
Datapoints are resampled with replacement using integer weights. Observations are resampled independently within each spectral-density frequency class. The total number of possible samples is finite and depends on the moiety type, number of fields, and (for deuterated methyl) field degeneracies. For large datasets this number may be prohibitively small — see [TODO: link to online docs] for details.

Bayesian
Datapoints are weighted using continuous samples drawn from symmetric Dirichlet distributions, one per spectral-density frequency class. Unlike nonparametric bootstrapping, this sampler has no upper limit on the number of samples.
